ASL Prediction

This project aims to predict the alphabet letter represented in an image of a hand sign from American Sign Language (ASL). The trained model predicts letters between A and I takes an input image and provides a prediction of the corresponding letter.

The given model is a CNN-based architecture for hand sign classification. It consists of three convolutional layers with increasing output channels, followed by max-pooling, batch normalization, and dropout layers. The model also includes two fully connected layers for classification, resulting in a total of six convolutional layers and two fully connected layers. It is too note that transfer learning was used and resulted in a more accurate model. However, that model was too large too implement on my version of Heroku.

This model was trainned on 11000 images from the following dataset:https://www.kaggle.com/datasets/lexset/synthetic-asl-numbers.
